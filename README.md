# Authentication with .Net Core 5

This is a simple project all-in for authenticate an user.

## Features

- Create user
- Retrieve user by username
- Login user

## Technologies

- ASP.Net Core 5
- Micro ORM Dapper
- Postgres Database
- JWT
- BCrypt

## Folders structure

- Authentication - (webapi project)
  - Controllers
  - Dtos
    - Request
    - Response
  - Infrastructure
  - Models
  - Repositories
    - Queries
  - Services
  - DB
